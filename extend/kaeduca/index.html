Kaeduca customization plugin

Este plugin esta pensado para customizar Goteo.

El funcionamiento está basado en la sustiticion de vistas.

Vistas:

Hay 3 tipos de vistas en goteo, las antiguas (legacy) y las nuevas (default y responsive).

Las nuevas vistas permiten una mayor flexibilidad a la hora de sustituir partes, se usa el sistema de vistas Foil (lectura recomendada si quereis cambiar muchas cosas).

Las vistas "legacy" no permiten gran cosa, únicamente sustituir enteramente la vista.

En general, para susittuir una vista solo hace falta crear el mismo archivo siguiendo la misma estructura de directorio dentro del plugin. Por ejemplo:

Vista original (legacy): Resources/templates/legacy/community.html.php Vista sustituta (si existe): extend/kaeduca/Resources/templates/legacy/community.html.php

Vista original (Foil/default, no responsive): Resources/templates/default/about/contact.php Vista sustituta (si existe): extend/kaeduca/Resources/templates/default/about/contact.php

Vista original (Foil/responsive): Resources/templates/responsive/project/home.php Vista sustituta (si existe): extend/kaeduca/Resources/templates/responsive/project/home.php

La idea es sustituir todas las vistas "legacy" por tipo "Foil/default" o "Foil/responsive". El paso a "Foil/default" es bastante simple pero require modificar un poco el controlador y la vista (la migración no está completamente acabada por falta de presupuesto, idealmente algun día se completará). El paso a responsive es progresivo pues requiere de nuevo diseño (se va implementando a medida que se necesita en Goteo).

Custom CSS/javascript:

El Css y javascript se compilan mediante la herramienta Grunt (ver http://goteofoundation.github.io/goteo/docs/install.html).

En el caso de kaeduca esta complicación simplemente consta de copiar los archivos de la carpeta Resources/assets/css a dist/assets/kaeduca/css

En el servidor actual de Godaddy he puesto un link simbolico de esa carpeta a la del plugin de manera que no hace falta compilar (siempre que no cambie la estructura del plugin).

En resumen:

Modificar los archivos:

kaeduca/Resources/assets/css/default.css (para vistas Foil/default) kaeduca/Resources/assets/css/responsive.css (para vistas Foil/responsive) kaeduca/Resources/assets/css/legacy.css (para vistas legacy)
